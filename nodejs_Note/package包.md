# package包

1. **CommonJS的包规范允许我们将一组相关的模块组合到一起，形成一组完整的工具**
2. 将模块放到一个文件夹就会形成一个包
3. CommonJS的包规范由包结构和包描述文件两个部分组成
4. **包结构：用于组织包中的各种文件**
5. **包描述文件：描述包的相关信息，以供外部读取分析**
6. 包实际上就是一个压缩文件，解压后还原为目录，符合规范的目录，应该包含以下文件：
   - **package.json**  **描述文件（必须）**
   - bin（文件夹）   可执行二进制文件（非必须）
   - lib           js代码
   - doc           文档
   - test          单元测试
7. json文件不能包含任何注释
